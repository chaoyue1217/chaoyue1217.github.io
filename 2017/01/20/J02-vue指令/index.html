发表于 2017-01-20 | 分类于 Vue

Vue指令

  • v-text ：绑定数据
  • v-html ：绑定数据
    • 和双花括号作用相同，区别是v-html可以识别标签，可以认为是innerText和innerHTML的区别
  • v-show ：值为true/false，控制元素的显示和隐藏，检测的是display属性
  • v-if ： 值为true/false，控制元素是否显示在页面上，该指令控制的是元素是否存在
  • v-else ： 配合v-if使用

  • v-for ：循环遍历

    • v-for除了可以迭代数组，还可以迭代对象和整数

  • v-bind ：用于动态绑定DOM元素的属性

    • 可以简写为“ : ”

  • v-on ：事件的监听器

  • v-model 一般用于控制表单数据的双向绑定

  • v-once ：只进行一次数据渲染
