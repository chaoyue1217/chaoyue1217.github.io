Vue Basis

发表于 2017-01-20 | 分类于 Vue

vue

  • vue是一套构建用户界面的渐进式框架
  • 核心
    • 数据驱动
    • 视图组件化
  • MVVM
  • View（页面dom） —- ViewModel（监控者） —- Model（数据）
  • 简单的vue实例

创建实例的步骤

  • new Vue()
  • 设置数据data
  • 挂载元素
  • 数据渲染
  • 每个Vue 实例都会代理其 data 对象里所有的属性
    • 只有这些被代理的属性是响应式的，在实例创建之后添加新的实例属性，不会触发视图更新。

常用参数

  • el ：挂载元素
  • data ：设置数据
  • filters ：过滤器

  • 需求 : 将数据转化成整数再进行渲染

  • computed：计算属性

  • methods：方法

  • 我们可以使用methods定义一些方法，供组件使用

  • watch：观察

  • 检测数据的变化，并执行相应的操作
