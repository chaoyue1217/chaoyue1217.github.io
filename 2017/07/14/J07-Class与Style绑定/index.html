Class与Style绑定

发表于 2017-07-14 | 分类于 vue--从零开始

数据绑定一个常见需求是操作元素的class列表和他的内联样式。因为他们都是属性，我们可以用v-bind处理他们：只需要计算出表达式最终的字符串。不过，字符串拼接麻烦又易错。因此，在v-bind用于class和style时，vue.js专门增强了它。表达式的结果类型除了字符串之外，还可以是对象或数组。
